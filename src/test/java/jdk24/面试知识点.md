## 常见考题

**1. Class-File API 与 ASM、Javassist 选型？

**答：**
| | Class-File API | ASM / Javassist |
|---|----------------|-----------------|
| 来源 | JDK 标准 API | 第三方字节码库 |
| 场景 | 读/写/转换 class 文件、工具链 | 成熟、生态大、框架广泛使用 |
| 优势 | 无额外依赖、与 JVM 格式同步 | 性能、文档、社区、复杂转换 |
| 选型 | JDK 内置工具、轻量分析 | Spring、Hibernate、CGLIB 等深度字节码生成 |

**2. Gatherer 和 `Collector` 区别？

**答：**
- **`Collector`**：**终端**操作，消费流元素产出单一结果（List、sum、分组）。
- **`Gatherer`**：**中间**操作，类似自定义 `filter`/`map`，可**逐元素变换并向下游发射 0~n 个元素**（如滑动窗口、批处理）。
- 流水线：Source → **Gatherer**（中间）→ **Collector**（终端）。
