## 常见考题

**1. 文本块缩进规则？最后一行换行是否保留？

**答：**
- **缩进**：编译器会去掉与闭合 `"""` **对齐的最少前导空白**（各行取最小缩进作为基准），保留相对缩进。
- **行尾 `\`**：续行到下一行，不插入换行。
- **最后一行换行**：若闭合 `"""` 单独一行，通常**包含**文本块末尾换行；若 `"""` 紧跟内容，可能不额外加换行——以编译结果为准，面试可说「注意闭合符位置影响末尾 `\n`」。
- 适合 SQL/JSON/HTML，编译期转为普通字符串，无运行时拼接开销。

**2. `yield` 和 `return` 在 switch 表达式中的区别？

**答：**
- **`yield`**：在 **switch 表达式** 的 `case` 分支块中**产出该分支的值**，不改变外围方法返回。
- **`return`**：直接**结束整个方法**并返回，不仅结束 switch 分支。
- 箭头 `-> expr` 单表达式分支不需要 `yield`；块分支需要 `yield` 或 `return`（若要从方法返回）。
