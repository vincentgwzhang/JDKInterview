## 常见考题

**1. `toList()` 与 `Collectors.toList()` 返回列表可否修改？

**答：**
- **`stream.toList()`**（JDK16+）：返回 **不可变** List，不能 `add`/`remove`/`set`。
- **`collect(Collectors.toList())`**：返回 **可变** `ArrayList`（实现依赖供应商，通常可修改）。
- 需要后续修改集合用 `toList()` + 再包一层 `new ArrayList<>(...)` 或直接用 `collect(toList())`。

**2. Records 能继承其他 record 吗？

**答：**
- **Record 不能继承** 普通类或其他 record（隐式 final）。
- 可 **实现接口**；可用 **sealed 层次**：`sealed interface Expr permits Add, Num` + `record Add(int a,int b) implements Expr`。
- 子 record 与 sealed 父类型配合是推荐模式，不是「extends record」。
